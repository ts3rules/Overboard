\# Overboard



Overboard is a local multiplayer party game developed in Unity using C#.



Inspired by classic party games such as Crash Bash and Mario Party, Overboard is a collection of original mini-games designed for up to four local players. The project is currently in active development and features one playable mini-game, with a second mini-game under development.

\### About



Knockout is a four-player arena game where players control rolling balls and attempt to knock their opponents from the platform. As the match progresses, sections of the arena collapse, making survival increasingly difficult until only one player remains.



The game can be played with up to four local players, with AI opponents automatically filling any empty player slots.
